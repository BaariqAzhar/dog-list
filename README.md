
# Dog List



Learn how to make REST API use Go (fiber, gorm)

## Tech Stack

**Backend** Go, Fiber, gorm

**Database** MariaDB (MySQL), PhpMyAdmin


## Run Locally

Clone the project

```bash
  git clone https://github.com/BaariqAzhar/dog-list.git
```

Go to the project directory

```bash
  cd go-list
```

Start the server

```bash
  go run main.go
```
